
## The files are my vim and gvim config
## To download and set as vim config
### MAC and Linux
#### mkdir .vim
#### cd .vim
#### git clone "https://github.com/ryanyun2010/.vim"
#### mv vimrc .vimrc
#### mv gvimrc .gvimrc
#### mv .vimrc ~/.vimrc
#### mv .gvimrc ~/.gvimrc
### Windows
#### cd
#### mkdir vimfiles
#### cd vimfiles
#### cd .vim
#### git clone "https://github.com/ryanyun2010/.vim"
#### rm vimrc
#### rm gvimrc
#### touch vimrc
#### touch gvimrc
